Nội dung tạo mã QR
🖼️
Bấm để chọn · kéo thả · hoặc dán (Ctrl+V) hình vào đây
Chưa chọn hình
QR nhúng trực tiếp dữ liệu hình ảnh — chỉ hợp với hình rất nhỏ. Hình lớn nên tải lên Drive/Box rồi dán link ở tab "Link".
📎
Bấm để chọn file · hoặc kéo thả vào đây
Chưa chọn file
QR nhúng trực tiếp nội dung file (data URL) — chỉ hợp với file rất nhỏ (≈ dưới 2KB). File lớn nên tải lên Drive/Box rồi dán link ở tab "Link".
